# Portfolio Monorepo

Production-grade modular frontend monorepo for portfolio sites.

## Tech Stack

- Vite 5.2.8 + React 18.2.0
- Three.js + @react-three/fiber for 3D
- GSAP + Framer Motion for animations
- TypeScript 5.8.3
- Playwright for testing

## Quick Start

```bash
pnpm install
pnpm dev
```
